# Tic Tac Toe

A simple **Tic Tac Toe** game built with **HTML/CSS/vanilla JavaScript** (no build step).

## Features
- 2-player mode
- Optional **vs Computer** mode (simple AI)
- Reset button

## Run locally
### Option 1: Open the file
Open `index.html` in your browser.

### Option 2: Use a local server (recommended)
From the repo root:

```bash
python3 -m http.server 5173
# then open http://localhost:5173
```

## Controls
- Click a square to place your mark
- Use **Reset** to start a new game
- Use **Mode** to switch between 2 players and vs computer
